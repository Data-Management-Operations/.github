# .github

This repository manages the public GitHub organization profile for Florida Gulf Coast University internal data management and platform repositories.

## Purpose

The `.github` repository provides a brief public-facing notice for visitors who reach this GitHub organization but do not have access to internal repositories.

Most repositories in this organization are intended for authorized Florida Gulf Coast University personnel and approved collaborators only.

## Public Organization Profile

The public organization profile content is located here:

```text
profile/README.md
```

GitHub uses that file to display the public-facing organization overview.

## Access Notice

If you expected to have access to a repository in this organization, please request access through the appropriate Florida Gulf Coast University channels.

Access must be approved by the appropriate supervisor, project sponsor, data owner, system owner, or platform owner.

## What Belongs Here

This repository may include:

* Public organization profile content
* General access guidance
* High-level public-facing notices
* Non-sensitive information about repository access expectations

## What Does Not Belong Here

Do not store:

* Internal platform documentation
* Architecture details
* Source code
* Snowflake scripts
* Pipeline code
* Secrets
* API keys
* Passwords
* Tokens
* Connection strings
* Internal system names or configuration details
* Institutional data
